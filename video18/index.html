margin and padding difference using border

After applying border and bg color to the content then bg color is only applied iside the border but not outside the border
space b/w content to border is called padding
space b/w border and ouside the border is margin

total height

When ever we give height as 200px to an element like div then that 200px height will only assign to the element's content! The content will be 200px
hence when caliculating total height we consider, element height + top padding height + bottom padding height + top border height + bottom border height + top margin height + bottom margin height
All of this would make up the total height

box sizing:border box

for the same height of 200px if we also add box-sizing: border-box; property then the 200px height will be distributed to content,padding and border but not to margin

this is important:for this property the height will not be alloted to the margin but only to content,padding and margin
